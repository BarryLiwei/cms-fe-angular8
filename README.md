# Angular8 for CMS

### Project Technical Structure
***
*  Angular8
*  ng-zorro-antd
*  Angular/router
*  Angular/http
*  Rxjs
*  WebSocket
*  Webpack4
*  Less

### Install

***
project address: (`git clone`)

```
git clone git@github.com:xpioneer/cms-fe-angular8.git
```
install node_modules with `yarn`

```
yarn #in your command terminal
```
***
### Run

start server: (http://localhost:9100)

```
yarn run start
```

publish production

```
yarn run build
```
