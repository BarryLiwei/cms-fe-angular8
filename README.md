Angular5 for CMS

### 项目技术架构
***
*  angular5
*  ng-zorro-antd
*  angular/router
*  angular/http
*  rxjs
*  webpack
*  less

### 安装

***
项目地址: (`git clone`)

```
git clone git@github.com:xpioneer/cms-fe-angular5.git
```
通过`yarn`安装本地服务第三方依赖模块

```
yarn
```
***
### 运行

启动服务: (http://localhost:9100)

```
yarn run start
```

发布代码

```
yarn run build
```
